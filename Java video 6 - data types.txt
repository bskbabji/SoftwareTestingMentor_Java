Primitive data types
----------------------------
byte 8-bit
short 16-bit
int 32-bit
long 64-bit end with L
float 32-bit end with f
double 64-bit  end with d
boolean 1-bit
char 16-bit single character in single quote or ASCII value

byte, short , int and long store whole integer numbers


non-primitive data types (These are classes and have their own methods)
------------------------------
String
Arrays

we can create String object two ways.

String str = new String(); str = "My name is Swapna";

Other way is 
String str = "My name is Swapna";
